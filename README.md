# 居家管理 App

管理家中存貨與清潔工作頻率的 Web 應用程式。**純 HTML / CSS / JavaScript**，無需 build，無框架依賴。

## 功能

### 📦 存貨管理
- 分類管理：食品、日用品、清潔用品、藥品
- 細項記錄：品名、數量、單位、存放位置、最低庫存提醒、到期日
- 智慧篩選：依分類、存放位置、關鍵字搜尋，可篩選「需補貨」項目
- 快速編輯：卡片上直接修改數量與位置

### 🧹 清潔工作
- 頻率設定：每日、每週、每兩週、每月、每季或自訂天數
- 細項清單：每個任務可設定多個子項目
- 下次到期：自動計算下次清潔日期
- 一鍵完成：點擊「完成」更新上次執行時間

### 🏠 總覽儀表板
- 需補貨項目提醒
- 逾期清潔任務
- 即將到期清潔
- 基本統計

## 使用方式

```bash
# 本地預覽（需安裝 Node.js）
npm run dev

# 部署至 GitHub Pages
npm run deploy
```

或直接以瀏覽器開啟 `web/index.html` 即可使用。
